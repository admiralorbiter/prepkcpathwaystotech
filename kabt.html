Project Description

Kansas Association of Biology teachers asked Pathway students to redesign their organizations website and perform optimizations to speed up user experience.

Project Details

  • Speed up load times from an average of 3 seconds to 1 second
  • Redesigned front page and made various design changes around user experience
  • Restructured site to help with user experience and make more intuitive
  • Created documentation to help KABT staff update and make changes to the website.

Skills Learned

  • Agile Process via Trello Boards
  • Wordpress Frontend/Backend
  • CSS
  • Database Permissions
  • Creating technical documentation
