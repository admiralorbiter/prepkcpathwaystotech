Project Description

Kansas Association of Biology Teachers has had an established website for over ten years. Their content includes:

  • Multiple biology curriculum.
  • Valuable lessons for all sorts of science classes.
  • The Kansas Outstanding Biology Teacher Award process.

However, due to a lack of a dedicated webmaster, their site has slowed down in recent years. There were also issues of ease of access to change things on the site and security issues identified due to old frameworks.

The first tasks students tackled were securing the website and backing up the blog posts and biology curriculum. Then students handled speeding up the website significantly, working on a redesign to help with speed and user experience.

Project Details

  • Speed up load times from an average of 3 seconds to 1 second
  • Redesigned front page and made various design changes around user experience
  • Restructured site to help with user experience and make more intuitive
  • Created documentation to help KABT staff update and make changes to the website.

Skills Learned

  • Agile Process via Trello Boards
  • Wordpress Frontend/Backend
  • CSS
  • Database Permissions
  • Creating technical documentation
